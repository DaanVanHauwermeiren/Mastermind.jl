# Mastermind

[![Build Status](https://github.com/DaanVanHauwermeiren/Mastermind.jl/actions/workflows/CI.yml/badge.svg?branch=main)](https://github.com/DaanVanHauwermeiren/Mastermind.jl/actions/workflows/CI.yml?query=branch%3Amain)

A pure Julia implementation of the game Mastermind.

Current features: 

- play against a random generated solution
- supports custom number of pins, number of codes, number of guesses

- [ ] heuristic or other search algorithm to find a solution.
